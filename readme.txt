=== CCFOLIA Auto Roller v1.1 ===

이 확장 프로그램은 구글 스프레드시트에 작성된 캐릭터 시트의 능력치를
크롬 사이드 패널에 불러와서, 항목 클릭 한 번으로 코코포리아 채팅창에
주사위 명령어를 자동으로 전송해주는 도구입니다.

해당 프로그램은 현재 Chrome, Naver Whale 환경에서 작동하는 것을 확인하였습니다.

=============================================
[사용 방법]
=============================================
작성한 캐릭터 시트의 URL을 복사해, 링크 입력 창에 넣어주세요.
반드시, 캐릭터 시트가 들어있는 탭을 눌러 링크를 복사해야 합니다.
위와 같은 기능으로 캐릭터 시트가 하나의 문서에 여럿 들어있어도 사용 가능하게 되어 있습니다!

필요한 기능을 추가하고 삭제할 수 있으며, 수치를 조정하거나 카테고리를 조정할 수도 있습니다!

=============================================
[다이스 봇 선택]
=============================================
패널 상단의 [CC] / [CCB] 버튼으로 선택합니다.
CC  → CC<=수치 형식 (기본 CoC 봇)
CCB → CCB<=수치 형식 (보너스/패널티 다이스 포함)

선택한 봇은 자동 저장됩니다.

=============================================
[문제 해결 FAQ]
=============================================
Q. 버튼을 눌러도 채팅에 아무 반응이 없어요.
A. 사이드 패널을 열기 전 코코포리아 탭을 먼저 클릭하여 활성화 상태로
   만들어 두세요. 그런 다음 확장 아이콘을 눌러 패널을 여세요.
   만약 패널을 먼저 열어둔 경우에는, 패널을 접지 마시고 그 상태로 코코포리아 탭을 한번 새로고침 해주세요.

Q. "불러오기 실패"라고 뜨는데요.
A. 구글 시트를 '링크가 있는 모든 사용자 - 뷰어'로 공유했는지 확인하세요.
   비공개 시트는 접근이 불가합니다.
