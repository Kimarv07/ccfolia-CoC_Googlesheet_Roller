=== CCFOLIA Auto Roller v2.0 (사이드 패널 버전) ===

이 확장 프로그램은 구글 스프레드시트에 작성된 캐릭터 시트의 능력치를
크롬 사이드 패널에 불러와서, 항목 클릭 한 번으로 코코포리아 채팅창에
주사위 명령어를 자동으로 전송해주는 도구입니다.

=============================================
[포함 파일 목록]
=============================================
  manifest.json    — 크롬 확장 프로그램 설정
  background.js    — 아이콘 클릭 시 사이드 패널 오픈
  config.js        — ★ 스킬 그룹화 설정 (사용자 직접 편집)
  sidepanel.html   — 사이드 패널 화면 구조
  sidepanel.css    — 사이드 패널 디자인
  sidepanel.js     — 핵심 로직 (CSV 파싱, 그룹화, 전송)
  content.js       — 코코포리아 채팅창 자동 입력
  readme.txt       — 이 파일

=============================================
[1차 테스트 방법]
=============================================
1. 크롬 브라우저에서 chrome://extensions/ 입력 후 엔터.
2. 우측 상단 [개발자 모드] 스위치를 ON으로 켭니다.
3. [압축해제된 확장 프로그램을 로드합니다] 버튼 클릭.
4. 이 readme.txt가 있는 폴더를 선택합니다.
5. 코코포리아 방(https://ccfolia.com/rooms/...)에 접속합니다.
6. 브라우저 우측 상단 확장 프로그램 아이콘(퍼즐 조각)을 클릭합니다.
7. 목록에서 "CCFOLIA Auto Roller"를 클릭하면 사이드 패널이 열립니다.

=============================================
[구글 시트 연동 방법]
=============================================
1. 구글 스프레드시트에서 A열에 기능명, B열에 수치를 입력합니다.
   예시:
     A열            B열
     관찰력          60
     듣기            45
     근접전(격투)     70

2. [공유] 버튼 클릭 → '링크가 있는 모든 사용자 - 뷰어'로 설정 → 링크 복사.
3. 사이드 패널 상단 URL 입력창에 붙여넣고 [불러오기] 클릭.
4. 스킬 목록이 카테고리별로 표시됩니다!

=============================================
[스킬 그룹화 설정 방법 (config.js 편집)]
=============================================
config.js 파일을 메모장 등으로 열어서 categoryMapping을 수정하세요.

형식:
  "카테고리 이름": ["스킬명1", "스킬명2", ...]

규칙:
  - 스킬명은 구글 시트 A열과 완전히 일치해야 합니다. (공백, 괄호 포함)
  - 등록되지 않은 스킬은 자동으로 "기타" 그룹에 들어갑니다.
  - 수정 후 chrome://extensions/ 에서 🔄 새로고침 버튼을 누르세요.

=============================================
[다이스 봇 선택]
=============================================
사이드 패널의 [CC] / [CCB] 버튼으로 선택합니다.
  CC  → CC<=수치 형식 (기본 CoC 봇)
  CCB → CCB<=수치 형식 (보너스/패널티 다이스 포함)

선택한 봇은 자동 저장됩니다.

=============================================
[문제 해결 FAQ]
=============================================
Q. 버튼을 눌러도 채팅에 아무 반응이 없어요.
A. 사이드 패널을 열기 전 코코포리아 탭을 먼저 클릭하여 활성화 상태로
   만들어 두세요. 그런 다음 확장 아이콘을 눌러 패널을 여세요.

Q. "불러오기 실패"라고 뜨는데요.
A. 구글 시트를 '링크가 있는 모든 사용자 - 뷰어'로 공유했는지 확인하세요.
   비공개 시트는 접근이 불가합니다.
